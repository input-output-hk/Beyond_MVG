## Beyond MVG Application

The MS4 workshops were conducted on the Beyond MVG identify and
prioritise application:

- Live site: https://beyond-mvg.vercel.app
- Application source repository (separate from this data-import repo).

Recommendation backlog, RICE/MoSCoW assessments, comments and the
"Discussed in workshop" admin flags created during these sessions live
in the application database.
