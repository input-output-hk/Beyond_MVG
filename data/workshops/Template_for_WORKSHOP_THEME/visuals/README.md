# Visuals

Place Miro board exports and other visual artifacts here.

## Supported Formats

- `.png` -- for quick viewing on GitHub
- `.pdf` -- for print quality
- `.svg` -- for scalable/editable visuals

## Naming Convention

Use descriptive kebab-case names:

```
miro-board-name-section.png
workshop-results-cluster-1.png
```

## Export Instructions (Miro)

1. Open the Miro board
2. Select the area to export (or the full board)
3. Export as Image (PNG) or PDF
4. Place the file here and update the workshop README.md table
