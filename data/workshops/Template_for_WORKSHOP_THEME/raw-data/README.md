# Raw Data

Place CSV exports from Google Sheets here.

## Naming Convention

Use the original Google Sheet tab name, in lowercase kebab-case:

```
sheet-tab-name.csv
```

## Export Instructions

1. Open the Google Sheet
2. Go to the specific tab
3. File > Download > Comma Separated Values (.csv)
4. Ensure UTF-8 encoding
5. Place the file here and update the workshop README.md table
