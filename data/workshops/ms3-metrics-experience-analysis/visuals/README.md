# MS3 Visuals

Place Miro board exports (PNG or PDF) and dashboard screenshots used in
the MS3 governance metrics analysis workshop in this folder.

Use descriptive filenames, for example:

```
miro-overview-board.png
dashboard-drep-net-change.png
dashboard-gini-coefficient.pdf
```

Source links for the original Miro boards and the Beyond MVG dashboards
are listed in [`../sources.md`](../sources.md).
